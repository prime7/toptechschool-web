## Tech Stack

### Frontend
- Next.js
- TypeScript
- TailwindCSS
- NextAuth.js

### Backend
- Go
- Gin/Echo (Web Framework)
- JWT Authentication
- Supabase (Database)

### Infrastructure
- Vercel (Frontend Hosting)
- Railway/Cloud Run (Backend Hosting)
- Supabase (Database)
- GitHub Actions (CI/CD)
